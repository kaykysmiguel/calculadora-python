# 🧮 Calculadora em Python

Projeto desenvolvido para praticar lógica de programação utilizando Python.

## Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão

## Tecnologias utilizadas
- Python
- VS Code

## Objetivo
Treinar estruturas condicionais, variáveis e entrada de dados.

## Como executar

1. Instale o Python
2. Clone o repositório
3. Execute o arquivo:

python calculadora.py
